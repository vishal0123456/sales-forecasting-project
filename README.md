# Sales Forecasting Using Deep Learning Models

This project focuses on sales forecasting using Darts deep learning models for predicting future sales trends.

## Files
- darts-forecasting-deep-learning-global-models.ipynb
- dataset.csv

## Project Overview
This project uses time series forecasting techniques and deep learning models to analyze and predict sales performance.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Darts
- TensorFlow / PyTorch

## Features
- Sales trend analysis
- Time series forecasting
- Deep learning models
- Data preprocessing
- Visualization of results
